1. Create blank solution
2. Create new project (ASP.NET Core Web Application - blank project) inside the solution (01 JavaScript Introduction)
3. In the startup.cs, in the Configure method add app.UseStaticFiles(); before app.UseEndpoints
4. Create folder with name wwwroot in the project root folder
5. Create your html, css and jas files inside wwwroot folder
6. Run your application and access files by typing their name after the url in the browser window, e.g. https://localhost:44320/ => https://localhost:44320/my_file.html
